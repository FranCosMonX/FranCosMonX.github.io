Alterar as configurações de impressora pelo Editor de Politicas do Windows

Antes de iniciar, é importante ressaltar que o Editor de Políticas de Grupo (gpedit.msc) está disponível apenas nas edições Pro, Enterprise e Education do Windows 10 e Windows 11. Se você estiver usando a edição Home, essa ferramenta não estará acessível. Entretanto, se você visitar os Post's da Microsoft, verá que há métodos alternativos para acessar o Editor de Políticas de Grupo na edição Home, embora esses métodos não sejam oficialmente suportados pela Microsoft. Para saber mais sobre como, visite o seguinte o link a seguir: https://learn.microsoft.com/pt-br/answers/questions/4314147/como-instalar-o-gpedit-msc-no-windows-10-gpedit-ms.

  • Passo 1: Pressione as teclas Win + R para abrir a caixa de diálogo "Executar". Digite gpedit.msc e pressione Enter ou clique em "OK" para abrir o Editor de Políticas de Grupo Local.
  • Passo 2: No Editor de Políticas de Grupo Local, navegue até o seguinte caminho: Configuração do Usuário > Modelos Administrativos > Impressoras
